# Kanäle

## Einführung

In der Kanalverwaltung werden alle eingerichteten YouTube-Kanäle verwaltet.

![Kanäle](images/04_kanalliste.png)

## Funktionen

- Kanal anlegen
- Kanal bearbeiten
- Kanal löschen
- Synchronisieren
- Start-Assistent öffnen
- Playlist-Manager öffnen

## Tipps

💡 Jeder Kanal besitzt eigene Einstellungen und Downloadordner.
