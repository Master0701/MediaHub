# Die Benutzeroberfläche

[[IMAGE:01_startseite|Das MediaHub-Hauptfenster nach dem Start]]

Nach dem Start erscheint das Hauptfenster von MediaHub. Von hier aus erreichst du alle wichtigen Bereiche.

Das Fenster besteht aus diesen Hauptbereichen:

- Menüleiste oben
- Werkzeugleiste mit Schnellaktionen
- linke Navigation
- Arbeitsbereich in der Mitte
- Statusleiste unten

## Menüleiste

[[IMAGE:02_menueleiste|Die Menüleiste von MediaHub]]

Die Menüleiste befindet sich oben im Fenster. Dort findest du klassische Programmfunktionen wie Datei, Kanäle, Werkzeuge und Hilfe.

Typische Aktionen in der Menüleiste sind:

- Start-Assistent öffnen
- neuen Kanal anlegen
- Tool-Center öffnen
- Scheduler öffnen
- Recovery Center öffnen
- Hilfe-Center öffnen
- Log-Ordner öffnen
- Systeminformationen anzeigen

## Werkzeugleiste

[[IMAGE:03_toolbar|Die Werkzeugleiste mit Schnellfunktionen]]

Die Werkzeugleiste liegt direkt unter der Menüleiste. Sie enthält die wichtigsten Schnellfunktionen.

Dazu gehören zum Beispiel:

- Wizard / Start-Assistent
- neuer Kanal
- Kanal bearbeiten
- Tools
- Playlist-Manager
- Vorschau
- Videos auswählen
- Sync
- Download stoppen
- Hilfe

Die Werkzeugleiste ist für Funktionen gedacht, die du häufig brauchst.

## Linke Navigation

[[IMAGE:04_kanalliste|Navigation und Kanalliste]]

Links befindet sich die Navigation. Sie bringt dich zu den Hauptseiten von MediaHub.

Wichtige Seiten sind:

- Dashboard
- Kanäle
- Bibliothek
- Downloads
- Jobs
- Scheduler
- Statistik
- Recovery
- Assistent
- Plugins
- Health Check
- Werkzeuge
- Einstellungen
- Log
- Hilfe

Wenn du eine Seite auswählst, ändert sich der Arbeitsbereich in der Mitte.

## Dashboard

[[IMAGE:05_dashboard|Das Dashboard von MediaHub]]

Das Dashboard ist die Startübersicht. Es zeigt dir den Zustand von MediaHub auf einen Blick.

Dort findest du unter anderem:

- Schnellaktionen
- Statusinformationen
- Hinweise des Assistenten
- Datenbank- und Download-Informationen
- Empfehlungen für nächste Schritte

## Log

[[IMAGE:06_log|Das Log zeigt Meldungen und Fehler]]

Im Log siehst du, was MediaHub gerade macht. Wenn etwas nicht funktioniert, ist das Log meistens die erste Stelle, an der du nachsehen solltest.

Typische Log-Meldungen sind:

- Kanal wurde geladen
- Download gestartet
- Tool fehlt
- Fehler beim Auslesen
- Backup erstellt

## Statusleiste

[[IMAGE:07_statusleiste|Die Statusleiste unten im Fenster]]

Die Statusleiste befindet sich unten im Fenster. Sie zeigt kurze Meldungen zur letzten Aktion und zum aktuellen Zustand.

Beispiele:

- Bereit
- Einstellungen gespeichert
- Hilfe geöffnet
- Kanäle geladen
- Download gestartet
