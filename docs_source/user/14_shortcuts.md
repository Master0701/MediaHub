# Tastenkürzel

## Einführung

MediaHub unterstützt verschiedene Tastenkürzel, um häufig verwendete Funktionen schneller aufzurufen.

Die wichtigsten Tastenkombinationen sind nachfolgend aufgeführt.

---

# Allgemeine Tastenkürzel

| Tastenkürzel | Funktion |
|--------------|----------|
| **F1** | Hilfe-Center öffnen |
| **F5** | Aktuelle Seite aktualisieren |
| **Strg + N** | Neuen Kanal anlegen |
| **Strg + Q** | MediaHub beenden |
| **Strg + B** | Backup erstellen |
| **Strg + E** | Einstellungen öffnen |
| **Strg + L** | Log anzeigen |
| **Strg + Umschalt + H** | Health Check öffnen |

---

# Navigation

Die linke Navigation ermöglicht den schnellen Wechsel zwischen den wichtigsten Bereichen.

Dazu gehören unter anderem:

- Dashboard
- Kanäle
- Bibliothek
- Downloads
- Scheduler
- Plugins
- Hilfe

---

# Tipps

💡 Wer regelmäßig mit MediaHub arbeitet, spart mit den Tastenkürzeln viel Zeit.

---

💡 Besonders hilfreich sind:

- **F1**
- **F5**
- **Strg + N**

Diese werden im täglichen Betrieb besonders häufig verwendet.

---

# Hinweise
