# Menüleiste

## Einführung

Die Menüleiste enthält alle zentralen Funktionen von MediaHub.

Viele Funktionen sind zusätzlich über die Werkzeugleiste oder die linke Navigation erreichbar.

![Menüleiste](images/02_menueleiste.png)

## Datei

Im Menü **Datei** befinden sich grundlegende Programmfunktionen.

Dazu gehören:

- Start-Assistent
- Beenden

## Kanäle

Im Menü **Kanäle** werden Funktionen für die Kanalverwaltung angeboten.

Dazu gehören:

- Neuer Kanal
- Start-Assistent
- Kanal bearbeiten
- Kanal löschen
- Playlist-Manager

## Werkzeuge

Das Menü **Werkzeuge** enthält Arbeitsfunktionen und Verwaltungsbereiche.

Dazu gehören unter anderem:

- Tool-Center
- Tools prüfen
- Vorschau
- Videos auswählen
- Playlists aus Manager laden
- Kanal synchronisieren
- Dashboard
- Bibliothek
- Job-Queue
- Scheduler
- Recovery Center
- Plugin Center
- Download-Warteschlange
- Download abbrechen

## Hilfe

Im Menü **Hilfe** befinden sich Dokumentation und Diagnosefunktionen.

Dazu gehören:

- Hilfe-Center
- MediaHub Assistent
- Anleitung
- Erste Schritte
- Changelog
- Versionshistorie
- Log-Ordner
- Systeminformationen
- Health Check
- Über MediaHub

## Tipps

💡 Viele Menüfunktionen haben Entsprechungen in der Werkzeugleiste.

💡 Wenn du eine Funktion nicht findest, lohnt sich ein Blick in das Menü **Werkzeuge**.

## Hinweise

⚠ Einige Menüeinträge sind nur sinnvoll, wenn bereits ein Kanal angelegt wurde.
