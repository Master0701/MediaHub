# Vorschau

Die Vorschau lädt die Videoliste eines Kanals, ohne Daten dauerhaft zu speichern.

Die Funktion wird über den Button **Vorschau** in der Werkzeugleiste gestartet.

Der Fortschritt erscheint in der Statusleiste.

Es öffnet sich kein eigenes Fenster.
