# Synchronisierung

## Einführung

Bei der Synchronisierung liest MediaHub neue Informationen eines Kanals ein.

Dabei werden keine Videos heruntergeladen.

## Ablauf

1. Kanal auswählen
2. Auf **Sync** klicken
3. Neue Videos und Playlists werden ermittelt
4. Datenbank wird aktualisiert

## Ergebnis

Anschließend können neue Videos über die Videoauswahl heruntergeladen werden.

## Hinweise

⚠ Die Synchronisierung ersetzt keinen Download.
