# Plugins

MediaHub v1.0 enthält ein vorbereitetes Plugin-System.

Plugins liegen später im Ordner:

plugins/<plugin_name>/

Ein Plugin enthält mindestens:

- plugin.json
- main.py
- help.md optional

## .mhplugin

Eine .mhplugin-Datei ist später ein Plugin-Paket, das MediaHub installieren kann.

Die vollständige Plugin-Ausführung ist für MediaHub 2.0 geplant.
