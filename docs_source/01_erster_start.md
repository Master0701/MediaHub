# Erste Schritte

Willkommen bei MediaHub.

MediaHub hilft dir beim Verwalten von YouTube-Kanälen, Playlists und Downloads.

## Nach der Installation

Starte MediaHub über die Desktop-Verknüpfung oder über das Startmenü.

Beim ersten Start legt MediaHub automatisch die benötigten Ordner an:

- config
- tools
- downloads
- logs
- archive
- plugins
