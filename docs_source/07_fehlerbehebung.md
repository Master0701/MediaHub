# Fehlerbehebung

Wenn MediaHub nicht richtig startet, prüfe zuerst:

- Ist MediaHub korrekt installiert?
- Gibt es Schreibrechte im Installationsordner?
- Sind die Tools vorhanden?
- Gibt es Fehlermeldungen in logs?

## Datenbankfehler

Falls eine Meldung zur Datenbank erscheint, prüfe den Ordner config.
