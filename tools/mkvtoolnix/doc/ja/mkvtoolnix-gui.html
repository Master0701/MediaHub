2. 説明#

mkvtoolnix-gui(1) は Qt(tm) に基づいた mkvmerge(1) の GUI です。mkvinfo(1) と mkvpropedit(1) の機能も実装しており、同様に mkvextract(1) の機能実装も視野に入れています。すべての設定（ソースファイルやトラックのオプション等）は保存・復元できます。

.mtxcfg 拡張子のファイル名を列挙すると、GUI はそれらの設定ファイルを適切なツールで読み込みます。それ以外のファイル名は、現在のモードに従って、多重化のソースファイルとして、あるいは情報ツール、チャプターエディタ、ヘッダーエディタのいずれかで開かれます。現在のモードは --merge、--info、--edit-chapters、--edit-headers で変更できます。既定のモードは多重化用のファイル追加です。

注意: すでにアプリケーションが起動している状態で2つ目を立ち上げた場合、コマンドラインで指定したすべてのファイル名は既存のインスタンスに渡されて処理されます。

Option Description
--merge

以降に続くすべてのファイル名が現在の多重化ジョブへのソースファイルとして追加されます。これは既定のモードです。
--info

以降に続くすべてのファイル名が情報ツールで開かれます。

さらに、起動時には多重化ツールではなく情報ツールが選択されます。
--edit-chapters

以降に続くすべてのファイル名がチャプターエディタで開かれます。

さらに、起動時には多重化ツールではなくチャプターエディタが選択されます。
--edit-headers

以降に続くすべてのファイル名がヘッダーエディタで開かれます。

さらに、起動時には多重化ツールではなくヘッダーエディタが選択されます。
-h, --help

ヘルプを表示して終了します。

-V, --version

バージョン情報を表示して終了します。
